# Data Grid

Surveillance-style data table with sticky header, auto-scroll capability, sortable columns with type auto-detection, automatic pagination, and motion-animated row entry.

```tsx
import { DataGrid } from "@mattloyed/eva-ui";
```

## Props

<PropsTable
  props={[
    { name: "columns", type: "DataGridColumn[]", required: true, description: "Column definitions: { key, header, width?, align?, sortable?, type? }" },
    { name: "data", type: "Record<string, string | number>[]", required: true, description: "Row data array" },
    { name: "color", type: '"orange" | "green" | "cyan"', default: '"green"', description: "Table color theme" },
    { name: "autoScroll", type: "boolean", default: "false", description: "Enable continuous scroll (pauses on hover)" },
    { name: "scrollSpeed", type: "number", default: "30", description: "Scroll speed in pixels/second" },
    { name: "maxHeight", type: "string", default: '"400px"', description: "Container max height" },
    { name: "title", type: "string", description: "Title bar text" },
    { name: "showIndex", type: "boolean", default: "false", description: "Show row index column" },
    { name: "pageSize", type: "number", description: "Rows per page — undefined shows all rows" },
    { name: "className", type: "string", default: '""', description: "Additional CSS classes" },
  ]}
/>

<Callout type="info" title="COLUMN SORTING">
  Set `sortable: true` on columns to enable click-to-sort. Type auto-detection checks the first value: integers, floats, datetimes, or string fallback. Override with `type: "int" | "float" | "datetime" | "string"`. Click cycles: unsorted → ascending → descending → unsorted.
</Callout>
