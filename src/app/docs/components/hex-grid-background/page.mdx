# Hex Grid Background

SVG-based honeycomb hexagon grid pattern. Positioned absolute to fill its parent container. Used to simulate A.T. Field visual patterns.

```tsx
import { HexGridBackground } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`<div className="relative h-48 border border-eva-mid-gray">
  <HexGridBackground color="#FF9900" opacity={0.15} />
  <div className="relative z-10 flex items-center justify-center h-full">
    <span className="text-eva-orange font-mono text-sm">A.T. FIELD ACTIVE</span>
  </div>
</div>`}
>
  <div className="relative h-48 border border-eva-mid-gray">
    <HexGridBackground color="#FF9900" opacity={0.15} />
    <div className="relative z-10 flex items-center justify-center h-full">
      <span className="text-eva-orange font-mono text-sm">A.T. FIELD ACTIVE</span>
    </div>
  </div>
</ComponentPreview>
