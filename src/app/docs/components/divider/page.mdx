# Divider

Horizontal or vertical separator with optional centered label. Supports three color themes and three line-style variants (solid, dashed, dotted).

```tsx
import { Divider } from "@mattloyed/eva-ui";
```

## Props

<PropsTable
  props={[
    { name: "label", type: "string", description: "Optional centered label text displayed as [ LABEL ]" },
    { name: "color", type: '"orange" | "green" | "cyan"', default: '"orange"', description: "Color theme for line and label" },
    { name: "variant", type: '"solid" | "dashed" | "dotted"', default: '"solid"', description: "Line style variant" },

  ]}

<Callout type="info" title="VERTICAL MODE">
  When `orientation="vertical"`, the label prop is ignored. The divider renders as an inline-block border-left element that stretches to its parent height.
</Callout>
