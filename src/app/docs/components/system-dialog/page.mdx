# System Dialog

Modal dialog with hex grid background overlay. Severity levels add red borders, blinking warning icons, and hazard stripes. Uses `AnimatePresence` for smooth enter/exit transitions.

```tsx
import { SystemDialog } from "@mattloyed/eva-ui";
```

## Severity Levels

<Callout type="info" title="INTERACTIVE">
  Click each button to open the dialog. Click ACCEPT, DECLINE, or the overlay background to close.
</Callout>
