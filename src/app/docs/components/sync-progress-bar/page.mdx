# Sync Progress Bar

Block-based LCD progress bar with dynamic color thresholds. Blocks fill left-to-right with color transitions based on value. Used for pilot synchronization rates and system metrics.

```tsx
import { SyncProgressBar } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`<SyncProgressBar value={72} label="SYNC RATE" />`}
>
  <SyncProgressBar value={72} label="SYNC RATE" />
</ComponentPreview>

## Color Thresholds

The bar automatically changes color based on the value:

<ComponentPreview
  title="THRESHOLDS"
  code={`<div className="space-y-4">
  <SyncProgressBar value={35} label="CYAN (0-50%)" />
  <SyncProgressBar value={65} label="GREEN (50-80%)" />
  <SyncProgressBar value={88} label="ORANGE (80-95%)" />
  <SyncProgressBar value={97} label="RED + BLINK (95%+)" />
</div>`}
>
  <div className="space-y-4">
    <SyncProgressBar value={35} label="CYAN (0-50%)" />
    <SyncProgressBar value={65} label="GREEN (50-80%)" />
    <SyncProgressBar value={88} label="ORANGE (80-95%)" />
    <SyncProgressBar value={97} label="RED + BLINK (95%+)" />
  </div>
</ComponentPreview>
