# Breadcrumb

Navigation breadcrumb trail with `>>` separator, uppercase labels, and color-coded links. The last item renders as the current page.

```tsx
import { Breadcrumb } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`import { Breadcrumb } from "@mattloyed/eva-ui";

<Breadcrumb
  items={[
    { label: "NERV HQ", href: "/" },
    { label: "OPERATIONS", href: "/ops" },
    { label: "UNIT STATUS" },
  ]}
/>`}
>
  <Breadcrumb
    items={[
      { label: "NERV HQ", href: "/" },
      { label: "OPERATIONS", href: "/ops" },
      { label: "UNIT STATUS" },
    ]}
  />
</ComponentPreview>
