# Countdown Timer

LCD-style countdown timer with MM:SS:ms format, battery bar visualization, and urgency color transitions. Used for operation time limits and self-destruct sequences.

```tsx
import { CountdownTimer } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="5 MINUTES"
  code={`<CountdownTimer initialSeconds={300} />`}
>
  <CountdownTimer initialSeconds={300} />
</ComponentPreview>

## Short Timer (Urgency Colors)

<ComponentPreview
  title="30 SECONDS"
  code={`<CountdownTimer initialSeconds={30} />`}
>
  <CountdownTimer initialSeconds={30} />
</ComponentPreview>

## With Expiry Callback

```tsx
<CountdownTimer
  initialSeconds={60}
  onExpire={() => {
    console.log("TIMER EXPIRED — EXECUTE CONTINGENCY");
  }}
/>
```

<Callout type="info" title="COLOR TRANSITIONS">
  Timer color changes based on remaining time: **&gt;60s** LCD green, **10–60s** orange, **&lt;10s** red with increasing blink intensity. Battery bar has 20 segments with thresholds at 75% and 25%.
</Callout>
