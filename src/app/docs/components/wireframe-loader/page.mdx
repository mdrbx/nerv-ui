# Wireframe Loader

Wireframe octahedron spinner inspired by Ramiel (the 5th Angel). Pure SVG geometry with framer-motion draw-in animation and continuous 3D Y-axis rotation. Zero external 3D libraries.

```tsx
import { WireframeLoader } from "@/components";
```

<Callout type="info" title="SVG GEOMETRY">
  The octahedron is drawn as an isometric projection using SVG polygons and lines. The outer diamond, inner diamond, cross axes, and structural edges create a convincing wireframe 3D effect without any WebGL or Three.js dependency.
</Callout>
