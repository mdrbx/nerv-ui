# Card

Structural panel container for dense monitoring surfaces, plus a specialized rounded `video` variant for subject/video overlays. Use the angular variants for control-room chassis and the rounded one only when a surface should read like captured footage annotation.

```tsx
import { Card } from "@mattloyed/eva-ui";
```

<Callout type="info" title="TWO FAMILIES">
  `default`, `alert`, and `hud` are structural cards for panels and chassis. `video` is different on purpose: it belongs to the camera-overlay family and should be reserved for footage, subject lockups, and signal annotations.
</Callout>

<Callout type="warning" title="DO NOT ROUND THE WHOLE SYSTEM">
  The rounded edges in the anime are localized to camera/video plaques. Keep navigation, shells, and standard monitoring cards angular; use `variant="video"` only where the UI should feel attached to an image feed.
</Callout>
