# Gauge

Semicircular gauge meter with animated needle, arc fill, and tick marks. SVG-based with Framer Motion draw-in animation. Supports threshold warning and five color themes.

```tsx
import { Gauge } from "@mattloyed/eva-ui";
```

<Callout type="info" title="240° ARC">
  The gauge uses a 240° arc sweep (from 7 o&apos;clock to 5 o&apos;clock) with 13 tick marks. The needle and value arc animate on mount with staggered Framer Motion transitions.
</Callout>
