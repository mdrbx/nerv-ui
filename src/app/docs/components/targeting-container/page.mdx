# Targeting Container

Wrapper component with corner L-brackets and optional crosshair grid background. Gives any content a targeting reticle aesthetic.

```tsx
import { TargetingContainer } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`<TargetingContainer label="TARGET-A" color="orange">
  <div className="p-8 text-center text-eva-orange font-mono text-sm">
    CONTENT AREA — LOCKED ON TARGET
  </div>
</TargetingContainer>`}
>
  <TargetingContainer label="TARGET-A" color="orange">
    <div className="p-8 text-center text-eva-orange font-mono text-sm">
      CONTENT AREA — LOCKED ON TARGET
    </div>
  </TargetingContainer>
</ComponentPreview>
