# Toast Notifications

Non-blocking NERV telemetry-style alerts triggered via the `useToast()` hook. Five severity variants with auto-dismiss progress bar, monospace typography, and CRT glow effects. Uses `AnimatePresence` for smooth slide-in/slide-out transitions.

```tsx
import { EvaToastProvider, useToast } from "@mattloyed/eva-ui";
```

## Setup

Wrap your application with `<EvaToastProvider>` to enable toast notifications:

```tsx
// app/layout.tsx or root component
import { EvaToastProvider } from "@mattloyed/eva-ui";

export default function Layout({ children }) {
  return (
    <EvaToastProvider>
      {children}
    </EvaToastProvider>
  );
}
```

Then use the `useToast()` hook anywhere inside the provider:

```tsx
import { useToast } from "@mattloyed/eva-ui";

function MyComponent() {
  const { addToast } = useToast();

  return (
    <button onClick={() => addToast({
      message: "PATTERN BLUE DETECTED",
      variant: "critical",
      duration: 5000,
    })}>
      ALERT
    </button>
  );
}
```
