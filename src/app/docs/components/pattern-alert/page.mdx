# Pattern Alert

Angel detection alert display inspired by the NERV target classification system. Shows designation badge, pattern/blood type classification, and measurement scale bars.

<Callout type="info" title="ANIME REFERENCE">
  Based on the Angel detection displays showing "11th ANGEL", "PATTERN: BLOOD TYPE BLUE" with scale markings used during target analysis sequences.
</Callout>
