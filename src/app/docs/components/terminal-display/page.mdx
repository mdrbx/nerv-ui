# Terminal Display

Monospace terminal emulator with optional typewriter effect. Auto-scrolls to bottom. Shows a blinking cursor, optional line numbers, and a status bar.

```tsx
import { TerminalDisplay } from "@mattloyed/eva-ui";
```
