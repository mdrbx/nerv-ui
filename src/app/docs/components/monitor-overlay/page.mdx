# Monitor Overlay

Absolute background overlay for NERV/MAGI-style monitor surfaces. Instead of a neutral sci-fi pattern, it builds the kind of framing seen around control screens: scan rails, registration brackets, guide lines, readout plaques, and optional alert bands.

```tsx
import { MonitorOverlay } from "@mdrbx/nerv-ui";
```

<Callout type="info" title="NERV / MAGI INTENT">
  This is the preferred backdrop primitive for command screens, auth gates,
  incident terminals, and modal overlays that should feel closer to the control
  room language of Evangelion.
</Callout>

<Callout type="warning" title="ABSOLUTE FILL">
  `MonitorOverlay` fills its parent with `absolute inset-0`. Wrap it inside a
  `relative` container and place your actual UI above it with a higher `z-index`.
</Callout>
