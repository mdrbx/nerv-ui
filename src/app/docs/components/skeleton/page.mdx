# Skeleton

Terminal-first placeholder skeleton. Text now reads more like a paper scanner pass than a CRT glow, while block and circular variants switch to a matrix of small white cells for a denser NERV monitor feel.

```tsx
import { Skeleton } from "@mdrbx/nerv-ui";
```
