# Title Screen

Cinematic title card with serif typography and staggered motion animation. Three alignment modes for different visual effects. Inspired by the iconic episode title cards from the anime.

```tsx
import { TitleScreen } from "@mattloyed/eva-ui";
```

## Props

<Callout type="info" title="ALIGNMENT MODES">
  **center** — Title and subtitle stacked vertically, centered. **split** — Title top-left, subtitle bottom-right (cinematic). **random** — Characters scattered with deterministic offsets.
</Callout>
