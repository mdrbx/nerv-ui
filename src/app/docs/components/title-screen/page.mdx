# Title Screen

Stark title card component for the EVA side of the visual language. It uses sparse framing, serif-led typography, and restrained motion so it reads like an anime title card rather than a NERV HUD panel.

```tsx
import { TitleScreen } from "@mattloyed/eva-ui";
```

<Callout type="info" title="ALIGNMENT MODES">
  **center** — Formal stacked title card. **split** — Episode-card layout with top-left lead and bottom-right dominant title. **random** — Controlled staggered assembly, still typographic rather than HUD-like.
</Callout>

<Callout type="warning" title="NOT A CONTROL-ROOM PANEL">
  `TitleScreen` is one of the few EvaUI components that should lean toward EVA’s ceremonial title-card language instead of the stricter NERV monitoring grammar.
</Callout>
