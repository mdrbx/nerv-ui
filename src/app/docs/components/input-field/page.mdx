# Input Field

Terminal-style text input with animated bracket decorators `[ ]` on focus. Shows error or hint text below. Extends all standard HTML input attributes.

```tsx
import { InputField } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`<InputField label="OPERATOR ID" placeholder="Enter ID..." color="orange" />`}
>
  <InputField label="OPERATOR ID" placeholder="Enter ID..." color="orange" />
</ComponentPreview>
