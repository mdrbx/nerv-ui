# File Upload

Drag-and-drop file upload zone with L-bracket corners, animated hover state, file list display, and built-in size validation.

```tsx
import { FileUpload } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`import { FileUpload } from "@mattloyed/eva-ui";

<FileUpload onFilesSelected={(files) => console.log(files)} />`}
>
  <FileUpload />
</ComponentPreview>

<ComponentPreview

<Callout type="info" title="FILE VALIDATION">
  Files exceeding the maxSize limit are rejected with an inline error message. Accepted files appear in a list below the drop zone and can be individually removed.
