# GradientStatusBar

A segmented horizontal bar with color-coded zones, a value indicator, and a bottom scale. Used for threat levels, resource meters, and any metric that spans multiple severity ranges.

```tsx
import { GradientStatusBar } from "@mdrbx/nerv-ui";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`import { GradientStatusBar } from "@mdrbx/nerv-ui";

<GradientStatusBar
  value={62}
  label="THREAT LEVEL"
  zones={[
    { start: 0, end: 33, color: "#10b981", label: "LOW" },
    { start: 33, end: 66, color: "#f59e0b", label: "MED" },
    { start: 66, end: 100, color: "#ef4444", label: "HIGH" },

/>`}

      { start: 66, end: 100, color: "#ef4444", label: "HIGH" },

</ComponentPreview>

    {

    {

      name: "fillMode",

    },
  ]}

<Callout type="info" title="ACCESSIBILITY">
  The bar renders with `role="meter"` and proper `aria-valuenow`,
  `aria-valuemin`, and `aria-valuemax` attributes for screen reader support.
</Callout>

<Callout type="info" title="HOVER LABELS">
  Each zone is hoverable and focusable. When a zone includes `label`, the bar
  switches to a pointer cursor and opens the NERV tooltip overlay with that
  label and its range.
</Callout>
