# Installation

EvaUI is a React component library inspired by the NERV/MAGI interfaces from Neon Genesis Evangelion. Built with Tailwind CSS, Framer Motion, and fully typed in TypeScript.

## Install the Package

```bash
npm install @mattloyed/eva-ui
```

<Callout type="info" title="PEER DEPENDENCIES">
  EvaUI requires **React 18+**, **Framer Motion 11+**, and optionally **Tailwind CSS 3.4+** (or 4+). Install any missing peer dependencies:
</Callout>

```bash
npm install react react-dom framer-motion
npm install -D tailwindcss
```


## Setup — Tailwind CSS 3

If you use **Tailwind CSS 3**, use the included preset:

```js
// tailwind.config.js
module.exports = {
  presets: [require("@mattloyed/eva-ui/tailwind.preset")],
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/@mattloyed/eva-ui/dist/**/*.{js,mjs}",
  ],
};
```

## Global Styles (Optional)

For the full NERV experience, add these global rules to enforce the brutalist aesthetic:

```css
/* Zero border-radius — brutalist industrial */
*, *::before, *::after {
  border-radius: 0 !important;
}

/* CRT Scanline overlay */
body::after {
  content: "";
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 9999;
  background: repeating-linear-gradient(
    0deg,
    transparent, transparent 2px,
    rgba(0,0,0,0.03) 2px, rgba(0,0,0,0.03) 4px
  );
}

/* Accessibility — respect reduced motion */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
  body::after { display: none; }
}
```

## Basic Usage

Import components directly from the package:

```tsx
import {
  EmergencyBanner,
  Button,
  TerminalDisplay,
  DataGrid,
  Gauge,
} from "@mattloyed/eva-ui";

export default function MyPage() {
  return (
    <div className="bg-eva-black min-h-screen p-8">
      <EmergencyBanner text="ALERT" severity="warning" visible />
      <Button variant="danger" size="lg">
        INITIATE OVERRIDE
      </Button>
    </div>
  );
}
```

## Available Components

EvaUI ships **32 components** covering layout, data display, forms, overlays, charts, and military-style HUD elements:

| Category | Components |
|----------|-----------|
| **Layout** | `TargetingContainer`, `HexGridBackground`, `Card`, `Accordion` |
| **Data Display** | `TerminalDisplay`, `DataGrid`, `SyncProgressBar`, `SyncRatioChart`, `SegmentDisplay`, `PhaseStatusStack`, `GradientStatusBar` |
| **Charts** | `BarChart`, `Gauge`, `PieChart` |
| **Navigation** | `NavigationTabs`, `EmergencyBanner` |
| **Forms** | `Button`, `InputField`, `SelectMenu` |
| **Overlays** | `SystemDialog`, `ClassifiedOverlay`, `TitleScreen`, `ToastProvider`, `StatusStamp` |
| **HUD / Military** | `TargetingReticle`, `SurveillanceGrid`, `PatternAlert`, `PilotCard` |
| **Special** | `MagiSystemPanel`, `CountdownTimer`, `SeeleMonolith`, `WireframeLoader` |

<Callout type="info" title="CAMERA OVERLAY SUB-FAMILY">
  `PatternAlert`, `PilotCard`, and bordered `StatusStamp` also belong to the dedicated camera-overlay grammar used for subject identification and signal-feed annotations. See <a href="/docs/getting-started/camera-overlays">Camera Overlay Grammar</a>.
</Callout>

<Callout type="warning" title="CLIENT COMPONENTS">
  All components use the `"use client"` directive. They are designed for React Client Components (Next.js App Router, Vite, Create React App, etc.).
</Callout>

## TypeScript

EvaUI is fully typed. All prop interfaces are exported alongside their components:

```tsx
import type { ButtonProps, DataGridColumn } from "@mattloyed/eva-ui";
```
